이 ADT는 두 정수를 입력받아 GCD(최대공약수)와 LCM(최소공배수)을 계산한다.  

- GCD 계산  
두 정수 모두 0이면 -1 반환 및 "GCD 정의 불가 (두 수 모두 0)" 출력 후 종료  
정수가 음수인 경우 양수로 변환  
유클리드 호제법으로 GCD 계산 및 출력  

- LCM 계산  
두 정수 모두 0인 경우 GCD 계산으로 -1 반환 및 "GCD 정의 불가 (두 수 모두 0)" 출력 후 종료  
두 정수 중 하나만 0인 경우 0 반환 및 출력  
나머지 경우 LCM 계산 공식으로 LCM 반환 및 출력  

- 정수 외 문자가 입력된 경우 "입력 오류: 숫자를 두 개 입력하세요." 출력 후 종료
